How A* works here

Each state = set of remaining topics. The algorithm expands states using f(n) = g(n) + h(n). g(n) is total adjusted time already spent (including cognitive-load penalties). h(n) is the admissible heuristic: sum of adjusted times for remaining topics, which never overestimates. A* guarantees the globally optimal ordering.
